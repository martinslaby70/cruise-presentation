# Plavba s chárby

https://martinslaby70.github.io/cruise-presentation/

Interactive cruise presentation — a visual slideshow pitching a Mediterranean cruise adventure to friends.

## Controls

| Key | Action |
|-----|--------|
| `→` | Next slide |
| `←` | Previous slide |
| `Space` | Advance step-by-step animations within a slide |

## Slides

1. **Title** — Plavba s chárby
2. **Intro** — Something for everyone (with ship animation)
3. **Water Activities** — Pools, slides, surfing, hot tubs, spa, sunbathing
4. **Itinerary** — Barcelona → Palma de Mallorca → Valencia → Málaga → Cádiz → Lisbon → Barcelona
5. **Food & Drink** — What's included + a sample day timeline
6. **Pricing** — Cabin tiers (Interior / Balcony / Suite) with what's included
7. **Casino** — Interactive slot machine mini-game
8. **Finale** — Closing slide
